1. Pengertian Inline Style Sheet

Inline Style Sheet merupakan fungsi CSS yang didefinisikan langsung pada tag HTML yang bersangkutan. Cara penulisannya cukup dengan menambahkan atribut " style ="..." " dalam tag HTML tersebut. Style hanya akan berlaku pada tag yang bersangkutan dan tidak akan memengaruhi tag HTML yang lain.

2. Pengertian Embeded Style Sheet

Embeded Style Sheet merupakan fungsi CSS yang didefinisikan terlebih dahulu dalam tag " " di atas tag " " atau lebih tepatnya di dalam tag " ".

3. Pengertian External Style Sheet

External Style Sheet merupakan fungsi CSS yang di defiisikan secara terpisah pada file yang berbeda. Selanjutnya, file atau halaman web yang ingin menerapkan style pada file CSS tersebut, hanya memanggil file CSS. Cara tersebut umumnya digunakan pada satu aplikasi berbasis web, sehingga tampilan antar halaman akan terlihat konsisten.
